# todosList

A To Do List application that keeps track of things you need to do!

## Built With

- HTML
- CSS/Bootstrap
- JavaScript
- jQuery
- Node.js
- Express.js
- PostgreSQL

![screenshot](images/todoList.png)

### Completed Features

- [x] Add to do's
- [x] Add priorities to to do's
- [x] Add categories to to do's
- [x] Add due dates to to do's
